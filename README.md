Nextjs + Jotai + Vanilla-extract + Styled-components 

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Storybook

```bash
npm run storybook
# or
yarn storybook 
```

## Test

```bash
npm run test
# or
yarn test
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
